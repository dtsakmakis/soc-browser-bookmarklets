Collection of lightweight browser bookmarklets designed to speed up common workflows for **security analysts, SOC operations, and everyday browsing automation**.

These bookmarklets run directly in the browser and **require no extensions or installations**.

---

# Live Demo

Try the bookmarklets directly in your browser:

👉 https://dtsakmakis.github.io/soc-browser-bookmarklets

You can **click**, **drag**, or **copy** bookmarklets directly from the page.

---

# Features

* 🔍 **IOC quick search**  
  (VirusTotal / AbuseIPDB / Whois / Hybrid Analysis)

* 🌐 **Bulk URL opener**

* 🌎 **Text translator**

* 🔢 **Consecutive numbers generator**

* 🟨🟥🟩 **Text highlighters**  
  Highlight text across the page (yellow / red / green)

* 🧹 **Highlight cleaners**  
  Remove highlights by color or clear all highlights

* 🔊 **Speech reader**  
  Read selected text aloud using browser speech synthesis

---

# Installation

## Option 1 — Drag & Drop (Recommended)

1. Open the live page:

https://dtsakmakis.github.io/soc-browser-bookmarklets

2. Drag any bookmarklet button to your **bookmarks bar**

3. Click the bookmarklet while browsing

---

## Option 2 — Click to Run

On the demo page you can also:

- Click any bookmarklet button
- It will run directly on the current page

---

## Option 3 — Copy Bookmarklet Code

If drag-and-drop is blocked:

1. Click **Copy Bookmarklet Code**
2. Create a new bookmark in your browser
3. Paste the code into the **URL / Location field**

# Bookmarklet Categories

## Investigation

**IOC Search**

Automatically detects the indicator type and opens relevant threat intelligence sources.

Supported IOC types:

- IP addresses
- Domains
- MD5 hashes
- SHA1 hashes
- SHA256 hashes

Sources opened:

- VirusTotal
- AbuseIPDB
- Hybrid Analysis
- Whois

---

## Utilities

- Bulk URL Opener
- Translator
- Speech Reader
- Consecutive Numbers Generator

Useful for:

- opening investigation links
- translating foreign content
- automating repetitive browser actions

---

## Page Highlighting

Highlight matching text across a page:

- Yellow Highlighter
- Red Highlighter
- Green Highlighter

Useful for:

- investigations
- page scanning
- identifying relevant content

---

## Highlight Cleaners

Remove highlights selectively or entirely:

- Yellow Cleaner
- Red Cleaner
- Green Cleaner
- All Highlights Cleaner

# Browser Compatibility

Works with most modern browsers:

- Chrome
- Edge
- Firefox
- Brave

Some browsers may block pop-ups the first time a bookmarklet opens multiple tabs.  
If this happens, allow pop-ups for the site.
