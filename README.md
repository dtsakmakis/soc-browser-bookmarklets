# Security Browser Bookmarklets

Collection of useful browser bookmarklets for security analysts and everyday browsing automation.

These bookmarklets perform quick actions directly from the browser without installing extensions.

## Features

- IOC quick search (VirusTotal / AbuseIPDB / Whois)
- Bulk URL opener
- Text translation
- Consecutive number generator
- Text highlighters (yellow / red / green)
- Highlight cleaner
- Speech reader

## Installation

1. Copy the JavaScript code from the bookmarklet file
2. Create a new bookmark in your browser
3. Paste the code in the URL field
4. Click the bookmarklet when browsing

## Example Use Cases

- SOC investigations
- Threat intelligence lookups
- Quick browser automation
- Page content highlighting

## Disclaimer

Only use bookmarklets from trusted sources as they execute JavaScript in your browser context.
