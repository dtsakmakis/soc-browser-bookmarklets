SOC Browser Bookmarklets

Security-focused browser bookmarklets for quick IOC lookups, automation, text utilities, and investigation workflow helpers.

View Repository Jump to Bookmarklets

How to Use

  • Make sure your bookmarks bar is visible.
  • Click a button to run it on this page.
  • Drag any button to your bookmarks bar to use it anywhere.

What This Includes

  • IOC and browser investigation helpers
  • Bulk URL automation
  • Text utilities and speech tools
  • Page highlighters and cleanup helpers

Important Note

Bookmarklets execute JavaScript in the context of the current webpage. Only use bookmarklets from trusted sources and test carefully.

Utilities

Click a button to run it on this page, or drag it to your bookmarks bar.

IOC Search

Threat Intel SOC IOC Lookup

Detects IPs, domains, and common file hashes, then opens relevant intelligence sources such as VirusTotal, AbuseIPDB, Whois, and Hybrid Analysis.

Click or Drag: IOC Search

Bulk URL Opener

Automation Browser Utility

Opens multiple URLs in separate tabs. Useful when you need to open a batch of cases, sources, dashboards, or reference links quickly.

Click or Drag: Bulk URL Opener

Translator

Text Utility Browser Helper

Translates selected text in a new tab using Google Translate. Useful for foreign-language pages, domains, comments, and messages.

Click or Drag: Translator

Speech Reader

Accessibility Text Utility

Reads selected text aloud using the browser speech synthesis API. Helpful for quick reviews of selected content.

Click or Drag: Speech Reader

Consecutive Numbers Generator

Utility Clipboard

Generates a range of consecutive numbers from input like 5-10 and copies the result to the clipboard.

Click or Drag: Number Generator
